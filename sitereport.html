Site Report

01 Introduction

This portfolio website was developed for my CSY1063 Web Development assignment at the University of Northampton. The objective was to create a fully responsive portfolio website using only HTML and CSS. The site serves as a platform to display my projects, skills, and contact information.

02 Planning & Design

The website follows a dark mode design, as I personally prefer dark themes for better readability and a modern look. The primary accent color used throughout the site is red (#e74c3c), which adds a bold and eye-catching contrast against the dark gray background. The text is styled in light gray/white to ensure high readability and a clean, professional appearance.

03 Development Process

The website consists of multiple pages, including index.html, project.html, contact.html, and sitereport.html. All CSS is stored in style.css for consistency and maintainability.

The navbar was implemented using flexbox for desktop layout, while a **CSS-only toggle menu** was used for mobile. The **Projects page** uses a CSS grid layout to display project placeholders. One of the challenges I faced was making the **mobile menu work without JavaScript**, which was solved using the :checked pseudo-class.

04 GitHub & Version Control

I used Git for version control and GitHub for Central code base. This helped me track changes, revert mistakes, and maintain a clean development history. My repository can be accessed at: GitHub Repository .

05 Testing & Validation

The website was tested for **cross-browser compatibility** on Chrome and Edge. The **Chrome Developer Tools** were used to check responsiveness across different screen sizes. The following validation tools were used:

  • W3C HTML Validator – Checked for valid HTML structure.
  • W3C CSS Validator – Ensured correct CSS syntax.

06 Conclusion

This project helped me boost my **HTML and CSS skills**, particularly in **responsive design**. In the future, I plan to enhance this portfolio by integrating **JavaScript** to improve interactivity and functionality.
